The Serply API is a tool designed for developers to access and manipulate search engine results programmatically.
It allows users to interact with various Google services, such as retrieving data from Google SERP (Search Engine
Results Pages), Google News, Google Scholar, Google Images, and more.

## Serply Handler Code
```python serply handler
from superagentx.handler.serply_api.serply_job_search import SerplyJobHandler

async def serply_job_handler():
    serply_job_handler = SerplyJobHandler(location='US')
    return await serply_job_handler.search('Java')
```
